Калькулятор чистой прибыли WB

Вносите товары Wildberries: артикул WB, цену продажи в рублях, курс RUB → KZT, себестоимость, упаковку, склад отправки, комиссию, логистику и налоги. Сайт считает чистую прибыль по каждой позиции в тенге.

Цена продажи вводится в рублях, расходы и чистая прибыль — в тенге. Комиссию можно подтянуть через WB API по артикулу WB: карточка даёт subjectID, а тарифы WB дают процент комиссии по выбранной модели продаж.

WB API для комиссии

Артикул WB → карточка товара → subjectID → комиссия по модели продаж
Для автозагрузки обычно нужен токен с доступом к разделам «Контент», «Цены и скидки» и «Тарифы». В одном HTML-файле токен хранится только у вас в браузере. Для стабильной работы API лучше запускать через proxy, потому что Wildberries может блокировать прямые запросы из браузера через CORS.
Введите API-ключ, артикул WB ниже и нажмите «Подтянуть из WB API».
